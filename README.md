Инструкция по запуску и установке:

Команда для запуска скрипта:

curl -sSL https://raw.githubusercontent.com/AppleXjack/autofront/refs/heads/main/setup-front.sh | sudo bash -s IP_BACKEND

IP_BACKEND - заменяем на IP сервера куда будет перегоняться траффик.

После завершения работы скрипта идем к вашему регистратору доменов и создаем домен/поддомен с IP вашего front сервера.

Далее при добавлении уже вашего основного сервера, соединение к которому будет через front - в вашей панели указываете домен, который назначен на front, но IP оставляете основного сервера.

Все работает.
